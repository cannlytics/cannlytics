How can you use Cannlytics?

Get Cannabis Data

Abundant, rich, valuable cannabis data is so close, yet so far away! Cannlytics puts these vital data in your hands by finding, wrangling, standardizing, and cleanly returning the data to you.

Get cannabis data →

The Cannlytics API

Do you have a technical staff? Than you can utilize the Cannlytics API to programmatically manage your operations and interface with your data. You have all of your data and the power of all laboratory software at your finger tips.

Read the Docs → 
{
  analyses: [...],
  lab_id: "cann-do",
  sample_name: "Super Easy",
  results: [...],
}

Plug & Play

Once you have your copy of Cannlytics, you can pick and choose engine components as you need them. Perhaps you need an addition to your existing software. Simply grab what you need and you're free to use it.

Use the SDK → 
# pip install cannlytics
from cannlytics.metrc import Metrc

# Connect to Metrc.
track = Metrc("vendor-api-key", "user-api-key")

# Get your lab results!
results = track.get_lab_results(uid="tag")
